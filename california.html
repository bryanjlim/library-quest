California Library Quest 📚

California residents can get library cards at any public library system in the state, not just their local branch. Each card adds that system's collection to the Libby app, opening up more ebooks, audiobooks, and waitlists. This map shows all 181 Libby systems.

Dot color = catalog. Libraries that share one Libby catalog share a color; independent libraries are grey.

Libraries with the same shared-catalog color share one Libby catalog and one holds list, so a card at any one gives the same books and waits — only one in the group is worth holding.

Independent libraries are grey: their own catalog and holds list, so each extra card means more books and shorter waits.

Each pin is a library system (one card per system), not an individual branch. Click a dot for details, or use the list checkboxes to collect; progress saves in your browser.
